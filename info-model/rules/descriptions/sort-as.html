<sort-as>

For elements that are sorted, the  <sort-as> element provides text that is combined with the base sort phrase to construct the effective sort phrase. The text can be specified in the content of the  <sort-as> element or in the @valueattribute on the <sort-as> element. The  <sort-as> element is useful for elements where the base sort phrase is inadequate or non-existent, for example, a glossary entry for a Japanese Kanji phrase.

Category: Utilities elements
